# 🎵 Sound Frequency Analyzer

This project analyzes sound using a microphone and finds:
- Frequency (Hz)
- Sound level (dB)

## Features
- Records sound
- Removes background noise
- Detects main frequency
- Shows graph

## How to Run
1. Install libraries:
pip install -r requirements.txt

2. Run code:
python main.py

## Output
- Frequency in Hz
- Volume in dB
- Graph of sound
